# Faker

## Описание
Библиотека для автоматической генерации объектов со случайными данными. Поддерживает рекурсию, настройку через **Expression Trees** и разрыв циклических зависимостей.

---

## Возможности
- **Базовые типы**: Числа (`int`, `double` и др.), `string`, `bool`, `char`, `byte`.
- **Системные типы**: `DateTime` (диапазон 2000г. — сегодня).
- **Коллекции**: Рекурсивное создание `List<T>`, `IEnumerable<T>`, `IList<T>`.
- **Конструкторы**: Приоритет конструкторам с макс. числом параметров; обработка исключений при вызове.
- **Циклы**: Автоматический разрыв циклических зависимостей (установка `null`).
- **Конфигурация**: Настройка полей через `FakerConfig` и деревья выражений (без строк).

---

## Структура
- **Faker**: Ядро библиотеки и генераторы.
- **Faker.Tests**: Модульные тесты на **xUnit**.

---

## Тестирование
В проекте реализованы проверки:
- **SimpleTypes**: Генерация примитивов и `DateTime`.
- **ComplexLogic**: Рекурсия, циклы, вложенные списки и выбор конструкторов.
- **Config**: Кастомные генераторы для свойств и параметров конструктора.
